                                                                                                                               # ShelfStack Library Management System

## الفروع (Branches):

### 1. فرع `main`
* **المكدس (Stack):** JavaScript.
* **التثبيت والتشغيل:**
  - للتشغيل: `node part_a.js` أو `node part_b.js`.

### 2. فرع `typescript`
* **المكدس (Stack):** TypeScript.
* **التثبيت والتشغيل:**
  1. انتقل للفرع: `git checkout typescript`.
  2. التثبيت: `npm install`.
  3. البناء: `tsc`.
  4. التشغيل: `node dist/part_a.js`.

 يرجى التبديل بين الفروع من القائمة في الأعلى لرؤية الكود الخاص بكل نسخة.*
